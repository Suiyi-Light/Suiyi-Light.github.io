网易云音乐API

发表于2020-09-25|更新于2020-12-25|API
|阅读量:

网易云音乐[ref]网易云音乐是一款由网易开发的音乐产品 https://music.163.com/[/ref] https://api.suiyil.cn/163-api/

代码源自 [github author=”injahow” project=”meting-api” /]

参数

type: 类型

name 歌曲名 artist 歌手 url 链接 cover 封面 lrc 歌词 single 获取以上所有信息(单曲) playlist 获取以上所有信息(歌单)

id: 网易云单曲ID或网易云歌单ID

使用参考

链接使用

歌曲url：https://api.suiyil.cn/163-api/?type=url&id=551339134 封面cover：https://api.suiyil.cn/163-api/?type=cover&id=551339134 歌词lrc：https://api.suiyil.cn/163-api/?type=lrc&id=551339134 歌词lrc（含翻译）：https://api.suiyil.cn/163-api/index_cn.php?type=lrc&id=551339134 歌曲信息single：https://api.suiyil.cn/163-api/?type=single&id=551339134 歌单信息playlist：https://api.suiyil.cn/163-api/?type=playlist&id=5225821464

配合APlayer[ref]🍭 Wow, such a beautiful HTML5 music player[/ref]和MetingJS[ref]🍰 A powerful plugin connect APlayer and Meting[/ref]

[alert]经后期测试该API与MetingJS配合的不是很完美。API写法与MetingJS不一致，可能部分功能无法使用，请自行测试调整。[/alert] 仅展示本站使用的吸底模式代码，更多使用方法见下方链接。

APlayer参数详见 https://aplayer.js.org/#/zh-Hans/ MetingJS用法详见 [github author=”metowolf” project=”MetingJS” /] 有什么问题欢迎留言询问
