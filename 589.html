Minecraft Java版 国际正版购买教程

 

Minecraft：Java版（Minecraft: Java Edition）是由Mojang AB开发的Minecraft的最初版本，适配于Windows、Mac OS与Linux。Notch于2009年5月10日开始游戏的开发，于2009年5月17日公开发布首个版本。完整的正式版游戏于2011年11月18日在MINECON 2011上发布。
——中文Minecraft Wiki:Java版

本文讲解的是Minecraft （我的世界）Java版 国际正版购买购买流程

即在minecraft.net中购买

如需使用我的世界中国版请前往 mc.163.com

1.准备材料

一个电子邮箱

一种支付手段（支付宝，信用卡等）

 

由于目前Minecraft.net使用谷歌reCAPTCHA验证码导致无法正常注册和登录，解决方法参考

https://www.mcbbs.net/thread-778175-1-1.html

 

2.进入官网

http://minecraft.net/

  • 点击获取MINECRAFT
选择电脑→PC→java版

 

  • 点击购买MINECRAFT
![](https://image.suiyil.cn/2020/07/mcbuy5-1024x569.png)

3.创建MOJANG 账户

# 由于 Minecraft Java 版购买已切换为微软账号，本文中关于 Mojang 账号注册内容失效，其他信息仅供参考

 

  • 点击留在 Minecraft.net
![](https://image.suiyil.cn/2020/07/mcbuy6-1024x492.png)
  • 按提示输入电子邮件地址、密码和生日等资料
  • 点击创建账户
 

由于目前Minecraft.net使用谷歌reCAPTCHA验证码导致无法正常注册和登录，解决方法参考

https://www.mcbbs.net/thread-778175-1-1.html

 

 

  • 输入验证码，重新登陆
![](https://image.suiyil.cn/2020/07/mcbuy7-1024x435.png)

4.填写信息，完成购买

  • 填写游戏名/游戏ID后选择适当支付方法付款后完成购买 ![](https://image.suiyil.cn/2020/07/mcbuy8.png)
  # 5.收取收据
  • 购买成功后，你的邮箱将会收到收据。
请妥善保存收据，如后期出现账户丢失等情况，可通过其中订单ID、交易ID联系MOJANG找回。

至此，Minecraft Java版 国际正版购买购买流程结束。
