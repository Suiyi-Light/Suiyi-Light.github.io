WordPress插件更新失败？试试WP-China-Yes 插件一键加速升级！

前言

相信在国内使用WordPress的站长在安装或更新时都遇见过安装失败的情况。需要一个可供使用的国内源，最近我发现了WP中国本土化社区（WP-China.org）及其WP-China-Yes 插件

WP中国本土化社区（WP-China.org），立志于建设国内的WordPress生态基础设施以求内化WordPress，使之更契合中国人的使用习惯。

插件功能介绍

v3.2.0特性

  1. 极致编码，安装包仅仅3KB大小，对服务器无任何负担
  2. 为官方应用市场提供反代加速，解决应用安装/更新缓慢、失败等问题
  3. 通过加速API调用和接管WordPress核心所依赖的静态文件来极大的加快站点后台访问速度
  4. 对接本土翻译平台，自动从本土翻译平台接收应用市场中所有作品的全量汉化支持（技术试验）
  5. 提供对Gravatar头像的加速支持
  6. 提供对谷歌字体的加速支持
  7. 提供对谷歌前端公共库的加速支持

插件下载

WP中国本土化社区下载地址 GitHub [github size=”mini” author=”wp-china” project=”wp-china-yes” /] 关于这个项目更多内容见WP中国本土化社区（WP-China.org） 最后感谢作者的这个项目
